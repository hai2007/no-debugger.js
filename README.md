## 如何本地调试?
首先你需要安装必要的包:

```bash
npm install
```

然后,启动服务器:

```bash
npm run server
```

最后,在浏览器中打开调试界面:

```
http://127.0.0.1:8080/test.spec.html
```

## How to use?
首先你需要通过命令行安装：

```bash
npm install --save no-debugger
```

安装好了以后，然后引入后就可以使用了：

```js
import 'no-debugger';
```

或者,你也可以通过CDN引入:

```html
<script src="https://cdn.jsdelivr.net/npm/no-debugger"></script>
```
